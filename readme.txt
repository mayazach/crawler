ΜΕΤΑΓΛΩΤΙΣΗ:
make -f crawler_make.mk
make -f server_make.mk

INPUT:
Το webcreator.sh δεχεται τα ορίσματα που περιγράφονται στην εκφώνηση με την ίδια σειρά
Τα myhttpd και mycrawler δεχονται τα ορίσματα που δίνονται στην εκφώνηση με οποιαδήποτε σειρά. Στο mycrawler όμως το starting_URL πρέπει να δίνεται τελευταίο

ΟΡΓΑΝΩΣΗ:
webcreator.sh: Δημιουργεί τα websites
myhttpd.c: Χωρίς threads - δέχεται GET requests και επιστρέφει ιστοσελίδες, δέχεται εντολές από το command port
mycrawler.c: Χωρίς threads - στέλνει αίτηση για να πάρει απο τον server τη σελίδα που δέχεται ως όρισμα και την αποθηκεύει
networking.c: οι συναρτήσεις για την επικοινωνία server και crawler - αποστολή και λήψη request-response-αρχείων
websites-save_dir: άδεια directories για τους webcreator-server και crawler αντίστοιχα

ΛΕΙΤΟΥΡΓΙΑ WEBCREATOR:

Δημιουργεί w sites με p html σελίδες στο φάκελο websites. Κάθε σελίδα περιέχει κείμενο, εσωτερικά και εξωτερικά links (τα links μέσα σε μια σελίδα είναι μοναδικά).
Ο φάκελος γίνεται purge πρώτα αν περιέχει αρχεία. Επίσης καλύπτονται οι ειδικές περιπτώσεις:

p=1,2: Επιτρέπεται μια σελίδα να έχει εσωτερικό link στον εαυτό της
w=2,p=1: Επιτρέπεται και τα 2 εξωτερικά links να αναφέρονται στην ίδια σελίδα

Μόνο η περίπτωση w=1 δεν είναι αποδεκτή καθώς δε γίνεται να υπάρχουν εξωτερικά links

ΛΕΙΤΟΥΡΓΙΑ SERVER-CRAWLER:

Ο server χρησιμοποιεί την select() για να ακούει στο port και στο command port ταυτόχρονα. Όταν έρχεται ένα request ελέγχει αν το πρώτο πεδίο του header είναι σωστό και αν τα υπόλοιπα υπάρχουν.
Στη συνέχεια ελέγχει αν το αρχείο που ζητήθηκε υπάρχει και μπορεί να ανοιχτεί και στέλνει το ανάλογο response header πίσω. Αν το αρχείο δεν ανοίχτηκε στέλνει το μήνυμα λάθους. Αλλιώς στέλνει το αρχείο.
Για να στείλει το αρχείο βρίσκει πρώτα το μέγεθος του αρχείου σε bytes και το στέλνει στον crawler, πριν στείλει το περιεχόμενο σε πακέτα μεγέθους 1024 bytes.

Όταν ο crawler παίρνει response 200 φτιάχνει το αρχείο που θα γραφτεί στον φάκελο save_dir. Υπάρχει φάκελος με αυτό το όνομα, μπορεί όμως να δοθεί κάποιο άλλο όνομα σαν όρισμα και θα δημιουργηθεί μαζί με τους υποφακέλους του.
